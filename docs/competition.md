# 最终测试说明

## 目标

自由组合模块和超参数，在测试集上达到尽可能高的平均 PSNR。

建议先完成必做模块和消融实验，再进入最终测试部分。

## 两个赛道

| 赛道 | 训练步数 | 重点 |
| ---- | ---- | ---- |
| Sprint | `100` | 快速收敛 |
| Standard | `500` | 最终质量 |

## 硬约束

以下设置不可修改：

| 项目 | 约束 |
| ---- | ---- |
| 高斯数量 | `200` |
| 背景色 | `(0.0, 0.0, 0.0)` |
| 随机种子 | `42` |
| 图像大小 | `128 x 128` |

## 可以调的内容

| 模块 | 可调内容 |
| ---- | ---- |
| Loss | 类型与超参数 |
| Initializer | 类型与超参数 |
| Optimizer | 类型与超参数 |
| Scheduler | 类型与超参数 |
| Param Groups | 各参数组 lr 倍率 |
| Model | 是否各向异性、是否启用 alpha |

## 测试集

最终测试使用 10 张测试图像，包括 5 张真实图像和 5 张合成图像。最终成绩取 10 张图的平均 PSNR。

注意：合成图像由 300 个高斯生成，而硬约束限定你只能使用 200 个高斯进行拟合。这意味着你无法完美复原目标，需要通过优化策略尽可能逼近。

## 评分

每个赛道单独按平均 PSNR 给分，满分 20 分。低于最低阈值不得分。

默认实现的平均 PSNR 大约为 Sprint `23.1 dB`、Standard `27.1 dB`。因此最低得分线设置在默认实现之上，直接提交默认配置不能获得最终测试分数。

### Sprint 赛道

| 平均 PSNR | 得分 |
| ---- | ---- |
| `>= 25.7 dB` | 20 |
| `>= 25.0 dB` | 16 |
| `>= 24.5 dB` | 12 |
| `>= 23.5 dB` | 8 |
| `< 23.5 dB` | 0 |

### Standard 赛道

| 平均 PSNR | 得分 |
| ---- | ---- |
| `>= 29.0 dB` | 20 |
| `>= 28.5 dB` | 16 |
| `>= 28.0 dB` | 12 |
| `>= 27.5 dB` | 8 |
| `< 27.5 dB` | 0 |

竞赛总分 = Sprint + Standard，满分 40 分，占总评 40%。不采用同学之间的排名，按固定 PSNR 阈值给分。

## 验收内容

- 报告中需单独汇报 Sprint 和 Standard 两个赛道的结果，并简要说明最终采用的设计及原因。
- 如果你新增了优化器、初始化器、loss 或 scheduler，需要在对应的 `build_*` 函数中完成注册。
- 代码提交要求详见 [README.md](../README.md) 中的提交要求部分。
