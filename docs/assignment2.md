# 任务 2 说明（30 分）

## 目标

自由组合模块和超参数，在测试图像上达到尽可能高的平均 PSNR。

建议先完成任务 1，再进入任务 2。

## 两种设置

| 设置 | 训练步数 | 重点 |
| ---- | ---- | ---- |
| 任务 2A | `100` | 快速收敛 |
| 任务 2B | `500` | 最终质量 |

## 硬约束

以下设置不可修改：

| 项目 | 约束 |
| ---- | ---- |
| 高斯数量 | `1000` |
| 背景色 | `(0.0, 0.0, 0.0)` |
| 随机种子 | `42` |
| 图像大小 | `128 x 128` |

## 可调整配置

除上述硬约束外，其余配置允许在合理范围内调整。可从以下几个维度进行组合或使用自己为任务2特别设计的实现：

| 配置项 | 可调整范围 |
| ---- | ---- |
| Loss | 损失函数类型及其超参数 |
| Initializer | 初始化策略及其超参数 |
| Optimizer | 优化算法及其超参数 |
| Scheduler | 学习率调度策略及其超参数 |
| Param Groups | 各参数组的学习率倍率 |
| Model | `use_anisotropic` 与 `use_alpha` 开关 |

## 测试图像

任务 2 使用 6 张测试图像，包括 3 张真实图像和 3 张合成图像。最终成绩取 6 张图的平均 PSNR。

| 编号 | 类型 | 说明 |
| ---- | ---- | ---- |
| R1 | 真实图像 | Flamingo（火烈鸟，色块简洁） |
| R2 | 真实图像 | Starry Night（星夜，绘画旋涡纹理） |
| R3 | 真实图像 | Parkour（跑酷，人物+建筑高频细节） |
| S1 | 合成目标 | Night Cityscape（城市夜景，823 高斯） |
| S2 | 合成目标 | Mandala（曼陀罗，1061 高斯） |
| S3 | 合成目标 | Coral Reef（珊瑚礁，1202 高斯） |

## 评分

每种设置单独按平均 PSNR 给分，满分 15 分。低于最低阈值不得分。

默认实现（random 初始化 + Adam）的平均 PSNR 大约为任务 2A `27.2 dB`、任务 2B `31.3 dB`。直接提交默认配置不能获得任务 2 分数。

### 任务 2A

| 平均 PSNR | 得分 |
| ---- | ---- |
| `>= 29.5 dB` | 15 |
| `>= 29.0 dB` | 12 |
| `>= 28.0 dB` | 9 |
| `>= 27.5 dB` | 5 |
| `< 27.5 dB` | 0 |

### 任务 2B

| 平均 PSNR | 得分 |
| ---- | ---- |
| `>= 33.5 dB` | 15 |
| `>= 33.0 dB` | 12 |
| `>= 32.0 dB` | 9 |
| `>= 31.5 dB` | 5 |
| `< 31.5 dB` | 0 |

任务 2 总分 = 任务 2A + 任务 2B，满分 30 分，占总评 30%。不采用同学之间的排名，按固定 PSNR 阈值给分。

## 验收内容

- 报告中需单独汇报任务 2A 和任务 2B 的结果，并简要说明最终采用的设计及原因。
- 如果你新增了优化器、初始化器、loss 或 scheduler，需要在对应的 `build_*` 函数中完成注册。
- 代码提交要求详见 [README.md](../README.md) 中的提交要求部分。
